## Installation

```bash
yarn install
```

## Environment

```bash
Create ".env.${mode}" file based on ".env.example"
Eg: ".env.development"
```

## Running the components
```bash
docker-compose up
```

## Running the app

```bash
# development
yarn dev

# production
yarn prod
```

## Test

```bash
# unit tests
yarn test

# e2e tests
yarn test:e2e

# test coverage
yarn test:cov
```
